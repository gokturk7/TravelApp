Proje Hakkında

Bu proje, React Native dersi laboratuvar çalışması kapsamında yapılmıştır. Uygulamada Stack ve Tab navigasyon yapıları kullanılarak sayfalar arası geçiş ve veri aktarımı sağlanmıştır.

Proje Yapısı

TravelApp/
├── App.js                  # Navigasyon yapılandırması
├── screens/                # Ekran dosyaları
│   ├── HomeScreen.js       # Ana ekran
│   ├── DetailsScreen.js    # Detay ekranı
│   └── SettingsScreen.js   # Ayarlar ekranı
├── package.json
└── README.md

Çalıştırma Talimatları
Proje klasöründe terminali açın ve sırasıyla şu komutları uygulayın:

Gerekli paketleri yükleyin:
npm install

Uygulamayı başlatın:
npx expo start
